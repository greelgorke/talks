Node.js is

  • Technologie
  • Plattform
  • Ecosystem
  • Community

als Technologie

"Node.js® is a platform built on Chrome's JavaScript runtime for easily building fast, scalable network applications. Node.js uses an event-driven, non-blocking I/O model that makes it lightweight and efficient, perfect for data-intensive real-time applications that run across distributed devices."

als Plattform

  • Fokus auf Netzwerkserver
  • bindings für System und C/C++ -libs
  • Modul System
  • Asynchrone Programmiermodell

als Ecosystem

  • npm rules the world, intentionally
  • one-thing-modules play together
  • cross-platform modules
  • C-lib bindings

als Community

  • 100k+ packages on npm
  • Back-Front-Full
  • CRubyJavaWebC#Python
  • GH-ML-Twitter-Blogs-Nodeschool
